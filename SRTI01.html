SRTI01 Manual

Overview:

You have 4 loops with a max length of 21 seconds of stereo audio each.
Input can be a stereo signal or 1/2 mono signals.
11 master effects and 3 loop effects are available to manipulate the sound.
Loops can be individually pitched up and down and reversed.
All effects can be randomized by holding down the Random button.
Each loop can be set to degrade over time.
Needs 9V DC power supply (center negative).
An optional momentary normally open foot pedal can be used for hands free recording or overdubbing.

Controls:

 Select loop A/B/C/D All loop specific controls work on the selected loop.
Loop volumes Control the volume of each loop individually. Unity indicated by long line.
Record Record on the selected loop.
The recording length determines the length of the loop.
Recording deletes any previous recording on that loop.
Overdub Overdub on the selected loop.
Overdub adds to the existing recording on that loop.
Resample Resample to the selected loop.
Records the current master output with any effects to the selected loop.
A short click will capture the previous loop length of the selected loop.
A long hold will capture the current output until the button is released. The hold time determines the new loop length.
After resampling all effects are reset to their default off position (all the way to the left or in the center).
Knobs will not respond until they have been turned back to default/off position. Then they can be activated again. If you resample to a loop with its gain turned down, effects will not be reset and you can continue to modify the sound.
Reverse Reverse the playback direction of the selected loop.
Degrade No effect at center. To the left the loop gets thinner and to the right it loses bits of sound over time.
Decay When set to on, the effect of the Degrade knob will persist on the loop and gradually degrade the sound over time. Decay is not active if the loop is being pitched (unless in Shift mode).
Pitch Pitch the selected loop up and down. Center is unity. Pitching changes the speed of the loop as well (e.g. pitching down 1 octave will take twice the time) if the Shift mode is off.
Shift When set to on, the Pitch knob will stop controlling speed and turn into a master pitch shifter. The output will be pitch shifted without changing speed and the shift will be applied to all loops currently playing.
Filter Apply a filter to the selected loop. Center is no effect. Turning left applies a lowpass filter, turning right applies a highpass filter.
Input Control the volume of the input.
Mute Input When set to on, the input will not be monitored in the output, however it will still be recorded when recording or overdubbing.
Tape Add tape saturation, filtering and pitch instability to the overall output.
Saturate Increase gain and add saturation and compression to the overall output.
In Stereo/Monomix When set to In Stereo, the input is stereo.
When set to Monomix, left and right input channels are mixed to mono, enabling 2 mono signals to be used together.
Rec Hold/Click When set to Rec Hold, recording and overdubbing happens while the Record or Overdub button is held down.
When set to Rec Click, recording and overdubbing starts and stops with each press of the Record or Overdub button. The pedal also follows this setting.
Pedal Rec/Dub Sets the function of an external foot pedal connected to the pedal input jack.
When set to Rec, the pedal controls recording on the selected loop.
When set to Dub, the pedal controls overdubbing on the selected loop.
Clear A short click will clear the selected loop.
A long hold will clear all loops. All loop lengths are set to the length of the hold.
Random Hold down to randomize all effect settings. When let go, the settings will return to current knob positions.
While holding randomize, effects can be changed by catching their randomized position with the knob. This allows you to keep all other randomized settings but change the pitch for example.
The randomized sound can be kept by resampling while holding the Random button. Let go of the random button when you have finished resampling. Be aware that any knobs not in their initial position will affect the sound after resampling.
Fragment To the left applies fragmentation to the spectrum like a bad signal. To the right the sound is fragmented in time like scrambled audio.
Failure To the left introduces mechanical failure resulting in wow and flutter. To the right introduces digital information failure resulting in noise and reduced fidelity.
Tone To the left low frequencies are enhanced and high frequencies are attenuated. To the right high frequencies are enhanced and low frequencies are attenuated.
Starve Simulates the effect of amplifiers starved for power, with 2 different modes to the left and to the right.
Octave To the left adds an octave down version to the signal. To the right adds an octave up version to the signal.
Delay Add a modulated delay. Knob sets dry/wet amount. Delay length is fixed. Feedback follows amount.
Grains Add a long granular delay. Knob sets dry/wet amount. Delay length is fixed. Feedback follows amount.
Reverb Add reverb. Knob sets dry/wet amount.

Comments:

When resampling, all the sound from all the loops and input is captured and recorded to the selected loop. If you want to resample only a specific loop, turn down the volume of other loops.

When using the short click resample, all changes made during the last loop length will be captured. If the loop pitch has been changed, the length of the resample will be based on the new pitch speed. If you do not want to hear effects being changed, wait for a full loop cycle before resampling.

If the input is muted and you start recording or overdubbing, the volume of the input is set by the loop volume sliders. This lets you record the loop without hearing it, until you bring up the loop volume.

When in Shift mode, Decay is still active even when pitching the loop.

Input is not amplified so low volume signals like microphones might need preamplification.

Loops are not saved when powering off the unit.

When resampling and starting the unit, certain internal effect parameters are randomized.

Only a momentary normally open foot pedal can be used. Using a normally closed pedal will cause the unit to malfunction.

The loops are free running and can only be synced through the timing of recording and resampling. More precise timing control can be achieved by using the rec click mode. All loops can be set to the same length by holding down the Clear button for the desired length and then only overdubbing on the loops. Loop lengths can doubled or halfed by resampling the loop with the Pitch knob set to 12 semitones down or up.
